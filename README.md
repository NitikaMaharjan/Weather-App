# SkyView Weather App

SkyView Weather App is a website built using **HTML, CSS, and JavaScript**. It fetches real-time weather data from a weather API and displays current conditions along with a 24-hour forecast.

## Live Demo

[https://nitikamaharjan.github.io/Weather-App/](https://nitikamaharjan.github.io/Weather-App/)

## Features

* Search weather by location
* Displays temperature, wind speed, humidity and chances of rain
* 24-hour weather forecast
